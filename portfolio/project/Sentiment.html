n8n Automation – Sentiment Analysis

Automated sentiment analysis workflow using n8n to extract online reviews, analyze emotions using AI, and deliver real-time insights through messaging platforms.

Modules

  • Chat Trigger (User Input)
  • URL Validation (Google Maps / Amazon)
  • Review Extraction using SerpAPI
  • AI Sentiment Analysis Engine
  • Result Delivery via Messaging (WhatsApp / Chat)

Overview

This project automates sentiment analysis using n8n workflows. It collects customer reviews from platforms such as Google Maps and Amazon, processes them using AI models, and classifies feedback into Positive, Neutral, or Negative sentiments — all without manual intervention.

Working

  • User sends a place or product URL via chat.
  • n8n validates whether the URL belongs to Google Maps or Amazon.
  • Reviews are fetched automatically using SerpAPI.
  • All reviews are combined and preprocessed.
  • AI model analyzes sentiments (Positive / Neutral / Negative).
  • Final sentiment summary is sent back to the user automatically.

Key Features

  • End-to-end automation using n8n
  • AI-powered sentiment classification
  • Supports multiple review platforms
  • Real-time results via messaging

Technologies Used

Automation: n8n
AI Model: Google Gemini Chat Model
APIs: SerpAPI
Messaging: WhatsApp / Chat Integration

Challenges & Solutions

Challenge: Handling different review sources and data formats.
Solution: Implemented URL validation and unified data processing before performing AI-based sentiment analysis.

Outcome / Impact

Enables businesses and users to quickly understand customer sentiment, saving time and supporting data-driven decisions through fully automated insights.
